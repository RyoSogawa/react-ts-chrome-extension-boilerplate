# react-ts-chrome-extension-boilerplate

Chrome Extension boilerplate with
- React
- TypeScript

## Prerequisites

* [node + npm](https://nodejs.org/) (Current Version)

## Includes the following

* TypeScript
* Webpack
* React
* Jest
* eslint
* prettier
* husky / lint-staged
* Example Code
    * Chrome Storage
    * Options Version 2
    * content script
    * count up badge number
    * background

## Setup
```
yarn
```

## Build
```
yarn build
```

## Build in watch mode
```
yarn dev
```

## Load extension to chrome

Load `dist` directory

## Test
```
yarn test
```
